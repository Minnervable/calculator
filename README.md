# calculator
This project is being done as part of The Odin Project curriculum
## Features
- Keyboard support!
- Backspace or undo button that removes the last thing on the screen
- Math chaining
- Negative number toggle button that flips the sign of any input
- Has all the functionality of a four function calculator!
## Button Legend
- All clear is Delete
- Change sign is Shift
- Enter is equals 
- The rest of the buttons are self explanatory
